# 图片资源说明

此目录用于存放个人介绍页面的图片资源。

## 准备替换的图片

| 文件名       | 用途       | 推荐尺寸    | 说明                                     |
| ------------ | ---------- | ----------- | ---------------------------------------- |
| `avatar.jpg` | 个人头像   | 300x300px   | 正方形图片，会自动裁切为圆形显示         |
| `bg.jpg`     | 全屏背景图 | 1920x1080px | 建议使用暗色调游戏相关图片，自动适配屏幕 |

## 替换步骤

### 1. 替换头像
将你的头像图片命名为 `avatar.jpg`（或 `.png`）放入本目录，然后在 `index.html` 中找到 `.avatar` 内的 SVG 占位代码，替换为：

```html
<img src="images/avatar.jpg" alt="头像" style="width:100%;height:100%;object-fit:cover;">
```

### 2. 替换背景图
将你的背景图命名为 `bg.jpg` 放入本目录，然后在 `css/style.css` 中找到 `.bg-overlay` 的背景样式，按照注释中的说明替换。
